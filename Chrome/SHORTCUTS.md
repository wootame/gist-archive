# Chrome ショートカットキーリファレンス

## タブとウィンドウ

**新しいタブを開く**
```
Ctrl + T
```

**新しいウィンドウを開く**
```
Ctrl + N
```

**シークレットウィンドウを開く**
```
Ctrl + Shift + N
```

**閉じたタブを再度開く**
```
Ctrl + Shift + T
```

**タブを閉じる**
```
Ctrl + W
```

**ウィンドウを閉じる**
```
Ctrl + Shift + W
```

**次のタブに移動**
```
Ctrl + Tab
```

**前のタブに移動**
```
Ctrl + Shift + Tab
```

**特定のタブに移動（1～8番目）**
```
Ctrl + 1～8
```

**最後のタブに移動**
```
Ctrl + 9
```

**タブを固定/固定解除**
```
Ctrl + クリック (タブ上で右クリック → 固定)
```

**新しいタブでリンクを開く**
```
Ctrl + クリック
```

**新しいウィンドウでリンクを開く**
```
Shift + クリック
```

**バックグラウンドタブでリンクを開く**
```
Ctrl + クリック
```

## ページ操作

**ページを再読み込み**
```
F5
```

**キャッシュをクリアして再読み込み**
```
Ctrl + F5
```

**ページ内検索**
```
Ctrl + F
```

**次の検索結果**
```
Ctrl + G
```

**前の検索結果**
```
Ctrl + Shift + G
```

**ページを保存**
```
Ctrl + S
```

**ページを印刷**
```
Ctrl + P
```

**PDF として保存**
```
Ctrl + P → 送信先を PDF に変更
```

**ページのソースを表示**
```
Ctrl + U
```

**ページ内の全選択**
```
Ctrl + A
```

```

**次のページに進む**
```

```

**ページの先頭にスクロール**
```
Home
```

**ページの最後にスクロール**
```
End

```

Shift + Space
```

**アドレスバーにフォーカス**
```
Ctrl + L

Ctrl + Enter (アドレスバー入力後)

**新しいタブでアドレスバーの内容を開く**
```
Alt + Enter (アドレスバー入力後)

## ブックマーク

**現在のページをブックマーク**
```
Ctrl + D
```

**全てのタブをブックマーク**
```
Ctrl + Shift + D
```

**ブックマークマネージャを開く**
```
Ctrl + Shift + O
```

## 検索・アドレスバー

**アドレスバーで検索**
```
Ctrl + K (または Ctrl + E)
```

**ページ内のテキストを検索**
```
Ctrl + F
```

**選択したテキストをGoogle検索**
```
選択 → 右クリック → Googleで検索
```

**選択したテキストを新しいタブで検索**
```
選択後に Ctrl + クリック (右クリックメニューから)
```
